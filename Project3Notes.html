Object Oriented Programming

What is Object Oriented Programming?
How is OOP valuable for large projects?
Object Oriented Programming is good for large projects because it allows a couple of key benefits
1. It allows programmers to associate functions in the program using plain conversational language. For example a function to build a web page can be titled make_html
2. Repative functionality can be grouped in common elements and called on multiple times to reduce coding repetition, making the program easier and faster to produce.
Connections between OOP and CSS & HTML using abstract thinking.
Much like CSS provides a high level framwork to use for HTML page to assist in formatting, classes in OOP can be used to define functionality that can be called on by specific instances in your Python program.

OOP Vocabulary

Object
A location in memory, set aside by a program, that has a value
Class
An extensible computer code template for creating objects.
Instance
A specific realization of an object in OOP
Method
the procedure associated with an object class.
Module
An independent interchangeable construct that contains everything necessary to execute one aspect of functionality in a program.
Library
a collection of implementations of behavior, written in terms of a language, that has a well-defined interface by which the behavior is invoked
Class Variable
A variable defined within a class for which a single copy exists
Inheritance
The mechanism that allows the attributes of the parent class to also flow down to the child class
Method Overriding.
A mechanism that allows the same named method in a child class to alter the value from the parent class
